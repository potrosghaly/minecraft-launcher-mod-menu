# Setup Guide

1. Download from [Releases](https://github.com/whiteblack-dunken3ho/minecraft-launcher-mod-menu/releases)
2. Extract anywhere (no admin rights needed)
3. Run `launcher.exe`
4. Click "New Profile"
5. Select version → adjust memory → launch

All extensions are loaded from the `extensions/` folder at runtime.  
No files are written to `.minecraft`.
